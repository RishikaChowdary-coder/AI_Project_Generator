# AI_Project_Generator
# AI Project Generator for Students

## Overview

AI Project Generator is a web-based platform that helps students generate innovative project ideas using Artificial Intelligence. It provides detailed project descriptions, technology stacks, implementation guidance, and downloadable project reports, making it easier for students to find suitable academic and hackathon projects.

## Features

### AI-Powered Project Generation

* Generates unique project ideas based on user interests.
* Provides project titles, descriptions, objectives, and features.
* Suggests suitable technologies and tools.

### Multiple Project Suggestions

* Generates multiple project ideas for a single query.
* Offers diverse options across domains such as AI, Web Development, IoT, Cybersecurity, and Mobile Applications.

### Save Projects

* Save favorite projects locally for future reference.
* Manage and review saved project ideas anytime.

### Download Project Reports

* Generate and download project details in a structured format.
* Useful for presentations, academic submissions, and hackathons.

### Responsive User Interface

* Modern and user-friendly design.
* Works across desktop and mobile devices.

### Fast AI Integration

* Uses Google's Gemini API to generate project content dynamically.
* Provides relevant and creative project recommendations.

## How to Run Locally

1. Clone the repository.
2. Navigate to the backend folder.
3. Install dependencies:

npm install

4. Create a .env file and add:

GEMINI_API_KEY=YOUR_API_KEY

5. Start the server:

node server.js

6. Open the frontend in your browser.

## Deployment

The application is deployed on Render and connected to GitHub for continuous deployment.
